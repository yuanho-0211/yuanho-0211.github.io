```c
#include<iostream>
#include<string>
using namespace std;
int main()
{
	string s;
	int k = 0;
	while (getline(cin, s))
	{
		
		for (int i = 0; i < s.size(); i++)
		{
			if (s[i] == '"')
			{
				if (k % 2 == 0)
				{
					cout << "``";
					k++;
					continue;
				}
				else 
				{
					cout << "''";
					k++;
					continue;
				}
			}
			else
			{
				cout << s[i];
			}
			
		}
		cout << endl;
	}
}
```
- 1 用數字去記"是第幾個，藉由奇數偶數判斷要改的
- 2 數字不能在讀取完一行後設=0，因為完整的輸入可能有兩行
###  3 是用string存，所以
```c
s[i] = '``';
s[i] = '''';
```
### 是錯的，因為s[i]是char。所以不能只改變需要改的s[i]，應該是邊讀邊改編輸出。
